OpenCV Assignment

This assignment includes basic image and video processing using OpenCV.

Tasks completed:

1. Reading and displaying image
2. Converting image to grayscale
3. Resizing image
4. Drawing shapes on image
5. Capturing video using webcam
6. Detecting edges in image
7. Recording and saving video

Tools used:

* Python
* OpenCV library

How to run:

* Install OpenCV using pip install opencv-python
* Run each Python file using python filename.py

Output:

* Images are displayed using OpenCV windows
* Video is recorded and saved as output.avi file

Conclusion:
This assignment helped in understanding image and video processing using OpenCV.
